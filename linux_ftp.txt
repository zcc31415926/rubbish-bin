linux 登录 ftp：
1.lftp username:password@address:port
2.lftp username@address; password （默认端口号为21）
3.lftp adddress; username; password
4.lftp; open address; username; password

linux ftp 中文乱码问题：
set ftp:charset gbk（或 gb2312 / utf-8）设置ftp的编码格式
set file:charset utf-8（或 gb2312）设置本地编码格式

linux ftp 查找文件：
ls directory file 显示ftp中directory文件夹中的所有文件
!ls directory file 显示本地directory文件夹中的所有文件
find directory -name "filename" 查找directory文件夹中带有filename的所有文件

linux ftp 下载文件：
lcd directory 设置本地存放目录为directory（默认为/home/usr）
get file 将文件file下载到本地存放目录
get -c file 将文件file断点续传下载到本地存放目录
mget file1 file2 ... 批量下载文件到本地存放目录
mget -c file1 file2 ... 断点续传批量下载到本地存放目录
mget -c directory/file 断点续传将directory目录下的文件file下载到本地存放目录
pget -c -n 10 file 以最多10个线程断点续传下载到本地存放目录
set pget:default -n x 设置下载最多线程数为x
mirror directory/ 将directory文件夹全部下载到本地存放目录并建立同名文件夹

linux ftp 上传文件：
put file 将file上传到ftp
mput file1 file2 ... 批量上传文件到ftp
mirror -R directory/ 将directory文件夹全部上传到ftp并建立同名文件夹
